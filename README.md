# SmartCart – E-Commerce Platform

SmartCart is a full-featured e-commerce platform inspired by marketplaces like Flipkart.  
It enables customers to discover products, manage carts and wishlists, make payments, and track orders, while providing admins with tools to manage the catalog, users, and sales operations.

## Core Features

### 👤 User Features
- User registration and login (JWT authentication)
- Product browsing with categories
- Search and filter products
- Add to cart / remove from cart
- Wishlist management
- Checkout with payment integration (Razorpay/Stripe demo mode)
- Order tracking
- User profile management

### 🛠 Admin Features
- Admin dashboard
- Add / edit / delete products
- Manage categories
- Manage orders
- View sales analytics

## Technologies Used
- **Frontend:** React.js (Hooks, Context API / Redux)
- **Backend:** Django + Django REST Framework
- **Database:** PostgreSQL
- **Authentication:** JWT
- **Payment:** Razorpay (Test Mode)
- **Deployment:** Render / AWS / Vercel
- **Version Control:** GitHub
